# ◆ QARA BAZAR

Маркетплейс локального творчества: художники, ремесленники, дизайнеры, локальные бренды и авторские проекты. От локального творчества — к международным.

Тёмная тема, этнический стиль (қошқар мүйіз, шанырак, восьмиконечная звезда), анимации, полностью рабочий бэкенд.

## Запуск

```bash
npm install
npm start
```

- **Витрина:** http://localhost:3000
- **Админка:** http://localhost:3000/admin

Порт можно поменять: `PORT=8080 npm start`.

## Админка

Логин по умолчанию: `admin` / `qara2026` — **смените пароль** в разделе «Настройки» после первого входа.

Через админку добавляются:
- **Товары** — с фото (до 8 штук, drag-and-drop), ценой, скидкой, остатком, категорией и мастером; флаги «Выбор базара» и «Показывать на витрине»
- **Мастера и бренды** — аватар, обложка, био, Instagram, сайт
- **Категории** — с этническими орнаментами-иконками
- **Заказы** — просмотр состава и смена статусов (новый → в обработке → отправлен → выполнен)
- **Настройки** — слоган, валюта, контакты, смена пароля

Товары не сидируются — витрина наполняется только вручную через админку.

## Покупатели

- **Регистрация и вход** — `/login` (email + пароль), сессия на 30 дней
- **Личный кабинет** — `/account`: профиль, история заказов со статусами и оплатой
- **Оформление заказа** — гостем или с аккаунтом (данные подставляются автоматически, заказ попадает в историю)
- **Оплата** — моковая платёжка «картой онлайн» (тестовый режим, деньги не списываются: любые 16 цифр) или при получении
- **Отзывы и рейтинги** — авторизованные покупатели ставят звёзды и пишут отзывы; средний рейтинг виден на странице товара; модерация в админке (разделы «Клиенты» и «Отзывы»)

## Стек

- **Backend:** Node.js (>=22.5), Express, встроенный `node:sqlite` (без нативных зависимостей), multer для загрузки фото
- **Frontend:** EJS (серверный рендер) + vanilla JS, без сборки
- **БД:** SQLite — файл `data/qarabazar.db` (создаётся автоматически)
- Корзина и избранное — в `localStorage`, заказы уходят в БД через API

## Структура

```
server.js          — Express-сервер
src/db.js          — схема БД, сид (админ, категории, настройки)
src/api.js         — публичное API (каталог, товары, заказы)
src/admin-api.js   — API админки (сессии, CRUD, загрузка фото)
src/pages.js       — серверный рендер страниц
views/             — EJS-шаблоны
public/            — стили, клиентский JS, админ-SPA, загруженные фото
data/              — SQLite-база
```
